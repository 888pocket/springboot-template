# springboot-template
Spring Boot Project Template
스프링 부트 프로젝트 템플릿

## 사용된 기술

- Java SDK 17
- Spring Boot 3.2.2
- H2

- [x] H2 연결
- [ ] CRUD api
- [ ] 동시성 테스트
- [ ] 유닛 테스트
- [ ] 통합 테스트
- [ ] e2e 테스트
- [ ] 로거
- [x] 커스텀 익셉션
- [x] 익셉션 핸들러
- [ ] jwt / 세션 기반 간단한 인증 암호화
